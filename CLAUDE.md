# DopaLock デモアプリ

## プロジェクト概要
DopaLock は、スマホを触らない時間を「レート・DP・友達バトル・店舗利用特典」に変える、大学生向けのスマホ封印型自習サービスです。

このリポジトリでは、**LEVEL UP STAGE 2026 の面接審査で見せるためのデモ Web アプリ**を作ります。
本番用ではなく、事業アイデアの体験価値を伝えるプロトタイプです。

## 技術スタック
- React + TypeScript
- Vite
- Tailwind CSS
- react-router-dom（画面遷移）
- lucide-react（アイコン）
- recharts（グラフ）
- localStorage（データ永続化）

## 開発方針
- 回答・説明は**日本語**で行う
- 実装前に必ず**短い計画**を出してから着手する
- 1 回の作業で大きく変更しすぎない
- 既存ファイルを大幅に壊さない
- UI は添付画像・`dopalock-demo/` 内のモックを参考にする
- 画像をそのまま背景に貼らず、**React コンポーネントとして再現**する
- 透過素材（`dopalock-demo/dopalock_assets_transparent/`）は**必要な箇所だけ**利用する（ロゴ・イラスト等）
- **デスクトップ表示を優先**する
- 面接で見せるため、**見た目・操作の分かりやすさ**を重視する
- バックエンド・認証は実装しない
- データは**モックデータ + localStorage** で管理する
- `git commit` は明示的に依頼されるまで行わない

## デザイン方針
- 白・ネイビー・薄いブルー・グレーを中心にする
- 清潔感・信頼感・SaaS 感を出す
- 学生向けサービスらしい、少しゲーム的な楽しさも入れる
- 角丸カード・薄い影・余白多め
- **左サイドバー + メインコンテンツ**の構成を基本にする

## 主要画面
- ホーム
- セッション作成
- 集中セッション中
- 5 分チケット休憩中
- セッション成功結果
- 脱獄結果
- ダッシュボード
- 履歴
- ランキング
- チケット
- ルール説明
- 設定

## 必ず動かす体験フロー
デモなので全ボタンが本番機能を持つ必要はない。ただし以下は必ず通しで動かす：
1. ホーム → セッション作成へ
2. 利用形態・モード・時間を選ぶ
3. セッション開始
4. 集中タイマー表示
5. 5 分チケット使用
6. 成功または脱獄
7. 結果画面に遷移
8. ダッシュボードに反映

## コマンド
```bash
npm install    # 依存インストール
npm run dev     # 開発サーバー起動
npm run build   # 本番ビルド
```
